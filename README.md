## Café del Rey — Next.js template

This repo is a **token-driven Next.js (App Router) design system** skinned for a coffee brand from **Los Yungas, La Paz (Bolivia)**.

Design reference: https://touchycoffee.com/

## Getting Started

### Local dev

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Base URL (metadata)

Set your canonical base URL via:

```bash
NEXT_PUBLIC_SITE_URL="https://your-domain.com"
```

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

### Theme

Default theme: `cafe-rey`

Night theme: `cafe-rey-night`
